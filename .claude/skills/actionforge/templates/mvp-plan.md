### Target user (ICP — Ideal Customer Profile)

[Specific description: role, industry, company size, or demographic. Not "developers" — "senior backend engineers at B2B SaaS companies with 20–200 engineers".]

[What outcome do they get that is worth paying for? What is the value in concrete terms?]

### Launch checklist

- [ ] Product works end-to-end for the core use case
- [ ] Onboarding flow is understandable without assistance
- [ ] Payment processing is functional (Stripe or equivalent)
- [ ] Error handling does not expose internal errors to users
- [ ] Basic analytics in place (sign-ups, activations, core action completions)
- [ ] Support channel set up (email, Discord, or Slack)
- [ ] Landing page clearly explains what the product does
- [ ] First 5–10 beta users identified and contacted
